# Production Security Checklist

- [ ] Replace all development secrets.
- [ ] Enforce HTTPS.
- [ ] Enable secure, HttpOnly, SameSite cookies.
- [ ] Require MFA for administrators.
- [ ] Disable default accounts.
- [ ] Restrict Keycloak admin console.
- [ ] Keep databases off the public internet.
- [ ] Keep Redis off the public internet.
- [ ] Review redirect URIs.
- [ ] Review web origins.
- [ ] Approve token lifetimes.
- [ ] Approve password policy.
- [ ] Enable brute-force protection.
- [ ] Test backups.
- [ ] Test restore.
- [ ] Configure monitoring.
- [ ] Configure alerting.
- [ ] Configure audit retention.
- [ ] Document signing-key rotation.
- [ ] Document incident contacts.
