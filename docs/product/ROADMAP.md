# Roadmap

## Phase 1

Foundation, documentation, local Compose, schema, API route foundation and admin UI shell.

## Phase 2

Complete Keycloak admin integration, service-token cache, JWKS refresh and real admin UI OIDC session handling.

## Phase 3

Invitation acceptance, email/GNS integration, session revocation, MFA policy projection and self-service portal.

## Phase 4

Integration examples, E2E tests, security scans, load tests and staging rollout.
