# User Personas

- Super admin: owns break-glass recovery, platform administrator assignment and high-risk global settings.
- Platform admin: manages users, applications, access grants and invitations.
- Security admin: manages MFA/security posture, sessions, identity providers, credential rotation and security review.
- Application admin: manages assigned application configuration, users, direct roles and permissions.
- Support operator: helps users with sessions, invitations and recovery without viewing secrets.
- Auditor: reviews audit and configuration history.
- End user: manages profile, sessions, MFA and accessible applications.
